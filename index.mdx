CS2 WebRadar is an open-source, browser-based radar for Counter-Strike 2. A lightweight C++ backend reads live game data directly from the CS2 process and broadcasts it over a local WebSocket server. The React web interface connects to that server and renders a live, interactive map — showing player positions, health, weapons, bomb status, grenade trajectories, and much more.

## How it works

CS2 WebRadar has two components that work together to deliver live game data to your browser.

<Steps>
  <Step title="Start Counter-Strike 2">
    Launch CS2 and load into a match or practice session. The C++ backend requires a running CS2 process to read game memory.
  </Step>
  <Step title="Run the backend">
    Execute `cs2_webradar.exe`. It reads player positions, weapon states, bomb data, and grenade locations from CS2's memory and streams updates over WebSocket on port `22006`.
  </Step>
  <Step title="Launch the web interface">
    Run `start.bat` (or `npm run dev` inside `webapp/`) to start the Vite dev server. Open your browser to `http://localhost:5173` to see the live radar.
  </Step>
  <Step title="Share (optional)">
    Click **Share Radar** in the top bar to copy a link containing your public IP. Send it to a friend — they can watch your game in their browser without installing anything.
  </Step>
</Steps>
